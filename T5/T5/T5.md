# Relatório do Projeto T5: Coloração do Mapa do Brasil

## Introdução
Este trabalho apresenta a aplicação do algoritmo **DSatur** (Degree Saturation) para o problema de coloração de vértices em um grafo que representa o mapa político do Brasil. O objetivo é obter uma coloração válida utilizando o menor número de cores possível através de uma heurística gulosa eficiente.

## Modelagem do Grafo
O mapa do Brasil foi modelado como um grafo não direcionado $G = (V, E)$, onde:
- **Vértices (V):** Representam os 26 estados brasileiros e o Distrito Federal (total de 27 vértices).
- **Arestas (E):** Representam as fronteiras terrestres entre os estados.

Os vértices foram indexados em ordem alfabética por sigla (0: AC, 1: AL, ..., 26: TO), conforme exigido pelo enunciado.

## Algoritmo DSatur
O DSatur é uma heurística que prioriza a coloração de vértices com maior **grau de saturação** ($DS(v)$), que é o número de cores distintas já presentes na vizinhança do vértice. Em caso de empate, utiliza-se o grau estático do vértice como critério de desempate.

### Passos do Algoritmo:
1. Colorir o vértice de maior grau com a cor 1.
2. Enquanto houver vértices não coloridos:
   - Selecionar o vértice $v$ com maior $DS(v)$.
   - Se houver empate, selecionar o de maior grau.
   - Atribuir a $v$ a menor cor disponível que não conflite com seus vizinhos.

## Resultados
O programa gera a lista de adjacência, a ordem de coloração, as cores finais de cada estado e valida se a coloração respeita a restrição de vizinhança.

## Conclusão
O algoritmo DSatur demonstra ser uma ferramenta poderosa para coloração de grafos planares, como mapas, frequentemente encontrando soluções próximas da ótima (número cromático) de forma muito rápida.
